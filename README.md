# Analisis-Statistik-Kemiskinan-Jawa-Barat-Tahun-2023
Project ini membahas analisis statistik data kemiskinan kabupaten/kota di Jawa Barat tahun 2023 menggunakan data dari Badan Pusat Statistik (BPS). Analisis dilakukan untuk mengetahui jumlah penduduk miskin, persentase kemiskinan, proporsi data, serta variasi tingkat kemiskinan antar daerah di Jawa Barat.
